# 試験対策項目

1. 試験のオフィシャルガイドとコースアウトラインを読んで、試験内容を理解します。
2. 合格に必要なトピックを特定します。
3. トピック毎に学習資料を選んで、勉強する予定を立てます。
4. 毎日の学習時間を決め、毎日勉強するように心がけます。
5. 定期的に模擬試験などを受けて、自分のレベルを評価します。
6. 弱点を特定して、より効率的に勉強するように調整します。
7. 最後には、試験前にリラックスして、試験当日に臨んでください。
